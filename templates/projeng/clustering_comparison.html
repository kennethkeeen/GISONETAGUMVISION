Analysis Summary

The comparative analysis was conducted using a dataset from ONETAGUMVISION project records, containing comprehensive features including spatial data (barangay, latitude, longitude), project characteristics (project_type, status, zone_type), and administrative boundaries.

Each algorithm was evaluated using multiple metrics including Silhouette Score (cluster quality), Zoning Alignment Score (administrative boundary compliance), Calinski-Harabasz Index (cluster separation), Davies-Bouldin Index (cluster compactness), and computational performance (execution time).

{{ best_algorithm|title }} achieved the highest Zoning Alignment Score, demonstrating superior performance in aligning with administrative boundaries, which is crucial for governance-oriented systems like ONETAGUMVISION.

The analysis shows that while data-driven algorithms (K-Means, DBSCAN, Hierarchical) effectively identify spatial patterns, Administrative Spatial Analysis proves most effective for governance-oriented clustering, achieving superior Zoning Alignment Scores and maintaining administrative compliance. To optimize both analytical performance and policy relevance, ONETAGUMVISION adopts a hybrid approach combining Administrative Spatial Analysis with GEO-RBAC, enabling clusters that are both data-driven and administratively compliant.
